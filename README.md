# ①課題名
エンジニアデータ管理マトリックス

## ②課題内容（どんな作品か）
- エンジニアのプロフィール情報（氏名、年齢、国籍、スキル、経験年数、稼働条件、希望単価など）を  
  データベースで一元管理するエンジニアデータ管理アプリ
- 登録・一覧表示・編集・削除（CRUD）が可能な管理画面を実装
- 実務でのSES／人材管理を想定したマトリックス型データ構造

## ③アプリのデプロイURL
- https://tokio-hirano-cyber.github.io/enginner-matrix/

※ GitHub Pages 上では UI デモ用途（HTML/CSS）のみ表示  
※ PHP / データベース処理はローカルまたはサーバ環境で動作

## ⑤工夫した点・こだわった点
- エンジニア情報を想定したデータベース設計（スキル・経験年数・単価など）
- POST通信を用いた安全なデータ登録処理
- 一覧画面を「マトリックス型」で視認性よく表示
- 管理画面として使いやすいモダンなUIデザイン（水色グラデーション）

## ⑥難しかった点・次回トライしたいこと（又は機能）
- PHP と MySQL を連携させたデータベース操作（CRUD）の実装
- SQL文とPHP側の変数管理の対応付け

### 次回トライしたいこと

- CSV / Excel へのデータ出力
- 案件データとのマッチング機能
